Universal Selector and Box-Sizing property

Box-sizing: content-box which is default or box-sizing

  • border-box: it includes the additional margin and padding value within the content's width. which is desirable in most case.
  • content-box: in this case, content-box width is fixed, if i add margin and padding on top of it, it will add up to the content's width and that will render on the screen.

universal selector *, use to reset default behaviors or to add some property to all elements to the page
